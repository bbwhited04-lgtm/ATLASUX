## Usage Guidelines

- All content is draft-ready for publication on the Atlas UX blog or external channels
- Adapt tone and length for the target platform (blog, LinkedIn, email newsletter)
- Update statistics and pricing references before publishing if values have changed
- All content follows Atlas UX brand guidelines: professional, honest, no manufactured hype
- Reference the Soul document principles: truth, accountability, long-term value

## Brand Voice

Atlas UX marketing content should be:
- **Direct**: Clear statements, no filler
- **Honest**: Real capabilities, no exaggeration
- **Professional**: Confident without being aggressive
- **Value-focused**: Lead with what the reader gains
- **Technical when appropriate**: Our audience appreciates substance over fluff
