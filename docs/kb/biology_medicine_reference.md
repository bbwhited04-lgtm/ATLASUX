## 1. Molecular Biology — The Central Dogma

### DNA → RNA → Protein
- **Replication**: DNA → DNA (semiconservative, 5'→3' on leading, Okazaki on lagging)
- **Transcription**: DNA → mRNA (RNA Pol II in eukaryotes, σ factor in prokaryotes)
- **Translation**: mRNA → Protein (ribosomes, tRNA, codons)
- **Reverse transcription**: RNA → DNA (retroviruses, telomerase)

### Genetic Code
- 64 codons: 61 sense + 3 stop (UAA, UAG, UGA)
- Start codon: AUG (methionine)
- Wobble position: 3rd base of codon
- Near-universal but not perfectly universal (mitochondria, some organisms differ)

### Gene Regulation
- **Prokaryotic**: Operons (lac, trp), σ factors, riboswitches
- **Eukaryotic**: Enhancers, silencers, transcription factors, chromatin remodeling
- **Epigenetics**: DNA methylation (CpG islands), histone modifications (acetylation, methylation)
- **Post-transcriptional**: miRNA, siRNA, alternative splicing, mRNA stability

### Key Enzymes
| Enzyme | Function |
|---|---|
| DNA Polymerase III | Main replication enzyme (prokaryotes) |
| DNA Polymerase I | Removes RNA primers, fills gaps |
| Helicase | Unwinds double helix |
| Topoisomerase | Relieves supercoiling |
| Ligase | Joins Okazaki fragments |
| Primase | Synthesizes RNA primers |
| Telomerase | Extends telomeres (reverse transcriptase) |
| RNA Polymerase II | Transcribes mRNA (eukaryotes) |
| Spliceosome | Removes introns from pre-mRNA |
| Ribosome | Translates mRNA to protein |

## 2. Genetics

### Mendelian Genetics
- **Law of Segregation**: Two alleles separate during gamete formation
- **Law of Independent Assortment**: Genes on different chromosomes assort independently
- **Dominance**: Complete, incomplete, codominance
- **Test cross**: Unknown genotype × homozygous recessive

### Non-Mendelian Inheritance
- **Linkage**: Genes on same chromosome tend to be inherited together
- **Crossing over**: Recombination frequency proportional to distance (1 cM ≈ 1% recombination)
- **Epistasis**: One gene masks another
- **Polygenic inheritance**: Multiple genes contribute to phenotype
- **Mitochondrial inheritance**: Maternal only
- **Genomic imprinting**: Parental origin determines expression

### Population Genetics
- **Hardy-Weinberg**: p² + 2pq + q² = 1 (genotype frequencies in equilibrium)
  - Requires: no mutation, random mating, no selection, large population, no migration
- **Genetic drift**: Random changes in allele frequency (stronger in small populations)
- **Natural selection**: Fitness differences drive allele frequency change
- **Mutation rate**: ~10⁻⁸ per base per generation (human)

### Human Genetics
- **Karyotype**: 46 chromosomes (22 autosome pairs + XX/XY)
- **Human genome**: ~3.2 billion base pairs, ~20,000 protein-coding genes
- **Trisomy 21**: Down syndrome
- **45,X**: Turner syndrome
- **47,XXY**: Klinefelter syndrome

## 3. Cell Biology

### Cell Structure
- **Prokaryotes**: No nucleus, circular DNA, 70S ribosomes, cell wall (peptidoglycan in bacteria)
- **Eukaryotes**: Membrane-bound nucleus, linear chromosomes, 80S ribosomes, organelles

### Organelles
| Organelle | Function |
|---|---|
| Nucleus | DNA storage, transcription |
| Mitochondria | ATP production (oxidative phosphorylation) |
| Chloroplast | Photosynthesis (plants) |
| ER (rough) | Protein synthesis, folding |
| ER (smooth) | Lipid synthesis, detoxification |
| Golgi | Protein modification, sorting |
| Lysosome | Degradation (pH ~5) |
| Peroxisome | Oxidation, H₂O₂ metabolism |

### Cell Cycle
- **G1**: Growth, prepare for S
- **S**: DNA replication
- **G2**: Growth, prepare for M
- **M**: Mitosis (prophase → metaphase → anaphase → telophase) + cytokinesis
- **Checkpoints**: G1/S (restriction point), G2/M, spindle assembly

### Cell Signaling
- **Signal transduction**: Ligand → Receptor → Second messenger → Cellular response
- **GPCR**: G-protein coupled receptors (largest receptor family)
- **RTK**: Receptor tyrosine kinases (growth factors)
- **cAMP pathway**: Adenylyl cyclase → cAMP → PKA
- **MAPK cascade**: Ras → Raf → MEK → ERK

## 4. Biochemistry

### Amino Acids (20 standard)
- **Nonpolar**: Gly, Ala, Val, Leu, Ile, Pro, Phe, Trp, Met
- **Polar uncharged**: Ser, Thr, Cys, Tyr, Asn, Gln
- **Positively charged (basic)**: Lys, Arg, His
- **Negatively charged (acidic)**: Asp, Glu

### Metabolism
- **Glycolysis**: Glucose → 2 pyruvate + 2 ATP + 2 NADH (cytoplasm)
- **Krebs cycle**: Acetyl-CoA → 2 CO₂ + 3 NADH + FADH₂ + GTP (matrix)
- **Electron transport**: NADH/FADH₂ → O₂ → ~30-32 ATP (inner membrane)
- **Total ATP per glucose**: ~30-32 ATP (aerobic)
- **Fermentation**: Pyruvate → lactate (anaerobic) or ethanol + CO₂ (yeast)

### Enzyme Kinetics
- **Michaelis-Menten**: v = V_max[S]/(K_m + [S])
- **K_m**: Substrate concentration at half V_max (measure of affinity)
- **Lineweaver-Burk**: 1/v = (K_m/V_max)(1/[S]) + 1/V_max
- **Competitive inhibition**: Increases apparent K_m, V_max unchanged
- **Noncompetitive inhibition**: Decreases V_max, K_m unchanged
- **Allosteric regulation**: Binding at non-active site changes activity

## 5. Immunology

### Immune System Components
- **Innate**: Barriers, phagocytes, NK cells, complement, inflammation
- **Adaptive**: T cells (cellular), B cells (humoral)
- **MHC I**: Present on all nucleated cells, present intracellular antigens to CD8+ T cells
- **MHC II**: Present on APCs (dendritic cells, macrophages, B cells), present to CD4+ T cells

### Antibodies (Immunoglobulins)
| Class | Function | Location |
|---|---|---|
| IgG | Main serum antibody, crosses placenta | Blood, tissue |
| IgM | First response, pentamer | Blood |
| IgA | Mucosal immunity, dimer | Secretions |
| IgE | Allergy, parasites | Mast cells |
| IgD | B cell receptor | B cell surface |

### Key Concepts
- **Clonal selection**: Antigen selects specific B/T cell clone for expansion
- **Somatic hypermutation**: Random mutations in antibody variable regions
- **Class switching**: Changing Ig constant region (IgM → IgG, etc.)
- **Central tolerance**: Deletion of self-reactive T/B cells during development
- **Peripheral tolerance**: Tregs, anergy, deletion of mature self-reactive cells

## 6. Neuroscience

### Action Potential
1. **Resting**: V_m ≈ -70 mV (K⁺ leak channels)
2. **Depolarization**: Na⁺ channels open, V_m → +30 mV
3. **Repolarization**: Na⁺ channels inactivate, K⁺ channels open
4. **Hyperpolarization**: K⁺ overshoot, V_m < -70 mV
5. **Refractory period**: Absolute (Na⁺ inactivated) then relative

### Neurotransmitters
| Neurotransmitter | Type | Function |
|---|---|---|
| Glutamate | Amino acid | Main excitatory (brain) |
| GABA | Amino acid | Main inhibitory (brain) |
| Acetylcholine | Ester | Neuromuscular junction, autonomic |
| Dopamine | Monoamine | Reward, movement |
| Serotonin (5-HT) | Monoamine | Mood, sleep |
| Norepinephrine | Monoamine | Alertness, sympathetic |

### Brain Regions
- **Frontal lobe**: Executive function, motor cortex, Broca's area (speech production)
- **Temporal lobe**: Auditory cortex, Wernicke's area (language comprehension), hippocampus (memory)
- **Parietal lobe**: Somatosensory cortex, spatial processing
- **Occipital lobe**: Visual cortex
- **Cerebellum**: Motor coordination, balance
- **Brainstem**: Vital functions (breathing, heart rate)

## 7. Ecology & Evolution

### Ecology
- **Trophic levels**: Producers → Primary consumers → Secondary → Tertiary
- **10% rule**: ~10% of energy transfers between trophic levels
- **r-strategy**: Many offspring, little parental care (insects, bacteria)
- **K-strategy**: Few offspring, high parental care (elephants, humans)
- **Competitive exclusion**: Two species cannot occupy same niche indefinitely

### Evolution
- **Natural selection**: Heritable variation → differential reproductive success
- **Genetic drift**: Random changes, stronger in small populations
- **Speciation**: Allopatric (geographic isolation) vs sympatric (same area)
- **Phylogenetics**: Molecular clocks, parsimony, maximum likelihood
- **Hardy-Weinberg equilibrium**: No evolution when all conditions met

## 8. Medicine Essentials

### Vital Signs (Normal Adult)
- Heart rate: 60-100 bpm
- Blood pressure: <120/80 mmHg
- Respiratory rate: 12-20 breaths/min
- Temperature: 36.5-37.5°C (97.7-99.5°F)
- SpO₂: 95-100%

### Common Lab Values
- Hemoglobin: 12-17 g/dL
- WBC: 4,500-11,000/μL
- Platelets: 150,000-400,000/μL
- Glucose (fasting): 70-100 mg/dL
- Creatinine: 0.7-1.3 mg/dL
- pH (arterial): 7.35-7.45
- Na⁺: 136-145 mEq/L
- K⁺: 3.5-5.0 mEq/L

### Pharmacology Basics
- **Agonist**: Binds receptor, activates response
- **Antagonist**: Binds receptor, blocks response
- **ED50**: Dose producing 50% of max effect
- **LD50**: Dose lethal to 50% of population
- **Therapeutic index**: TI = LD50/ED50 (higher = safer)
- **Half-life**: Time for 50% of drug to be eliminated
- **First-pass metabolism**: Oral drugs metabolized by liver before systemic circulation
