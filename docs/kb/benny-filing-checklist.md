# Trademark Filing Checklist Generator Knowledge

## Purpose

This document provides Benny (IP Counsel agent) with comprehensive knowledge for generating trademark filing checklists. A properly prepared application reduces the likelihood of Office actions, saves time and money, and maximizes the chances of smooth registration.

---

## Filing System Selection: TEAS Plus vs. TEAS Standard

### TEAS Plus ($250/class)

**Requirements:**
- [ ] All goods/services descriptions must be selected from the **USPTO ID Manual** (pre-approved descriptions only)
- [ ] Email address for correspondence provided
- [ ] Applicant agrees to receive all correspondence electronically
- [ ] All application requirements must be satisfied at the time of initial filing
- [ ] Correct filing fee paid at time of filing

**Advantages:**
- Lower cost ($250 vs. $350 per class)
- Pre-approved descriptions reduce the chance of identification-related Office actions

**Disadvantages:**
- Limited to ID Manual descriptions — if the applicant's goods/services don't match an ID Manual entry, TEAS Plus is unavailable
- If TEAS Plus requirements are not maintained, a $100/class surcharge converts the application to TEAS Standard

**When to recommend TEAS Plus:**
- The applicant's goods/services closely match one or more ID Manual entries
- Cost savings is a priority
- The application is relatively straightforward

### TEAS Standard ($350/class)

**Requirements:**
- [ ] Applicant may use free-form identification of goods/services
- [ ] Same general requirements as TEAS Plus (applicant info, mark, basis, etc.)
- [ ] Email address for correspondence

**When to recommend TEAS Standard:**
- The applicant's goods/services do not match any ID Manual entry
- A custom description is needed for comprehensive protection
- The goods/services are novel or specialized

## Required Application Fields

### 1. The Mark

**Standard Character Mark:**
- [ ] Mark text entered (letters, numbers, punctuation only)
- [ ] No design elements, no specific font/style/color
- [ ] "Standard character" claim checked
- [ ] Protects the word(s) in any font, style, size, color, or design

**Special Form (Stylized/Design) Mark:**
- [ ] JPG image uploaded (at least 250 pixels, no more than 944 pixels per inch)
- [ ] Image is clear, high-resolution, and accurately represents the mark as used
- [ ] If color is claimed: color claim statement included
- [ ] If color is claimed: description of the mark identifies where each color appears
- [ ] Mark description accurately describes all elements of the mark

**Sound Mark:**
- [ ] Audio file submitted (MP3, WAV, or OGG format)
- [ ] Detailed written description of the sound
- [ ] No drawing required (check "sound mark" option)

**Motion Mark:**
- [ ] Series of still images showing key frames of the motion
- [ ] Written description of the motion sequence
- [ ] Optional: video file

**Color Mark (Color Per Se):**
- [ ] Image showing the color(s) applied to the goods
- [ ] Color claim statement
- [ ] Detailed description of where colors appear
- [ ] §2(f) evidence (color marks almost always require acquired distinctiveness)

### 2. Applicant Information

- [ ] **Full legal name** of the applicant (individual name or entity name)
- [ ] **Entity type**: individual, corporation, LLC, partnership, joint venture, trust, estate, other
- [ ] **State/country of incorporation or organization** (for entities)
- [ ] **Domicile address** (physical street address — P.O. Box alone not acceptable for domicile)
- [ ] **Correspondence address** and **email address**
- [ ] **Foreign applicants**: U.S.-licensed attorney must be appointed (37 CFR §2.11(a))

### 3. Filing Basis

#### §1(a) — Use in Commerce

- [ ] Mark is currently in use in commerce (interstate, territorial, or between U.S. and foreign country)
- [ ] **Date of first use anywhere**: The date the mark was first used in connection with the goods/services, in any context
- [ ] **Date of first use in commerce**: The date the mark was first used in interstate/qualifying commerce
- [ ] **Specimen** showing the mark as used in commerce (one per class)
- [ ] Specimen meets all requirements (see Specimen section below)

**When to use §1(a):** The applicant is already selling the goods or rendering the services under the mark.

#### §1(b) — Intent to Use

- [ ] Applicant has a **bona fide intention** to use the mark in commerce
- [ ] Verified statement of intent signed by the applicant
- [ ] No specimen required at filing (specimen submitted later with Statement of Use)
- [ ] Be prepared to file a Statement of Use within 6 months of the Notice of Allowance (extendable to 36 months total)

**When to use §1(b):** The applicant has not yet started using the mark but has a genuine, good-faith plan to do so. Common for new product launches, rebrands, or pre-launch filings.

**Bona fide intent requirement:** Must be more than a mere subjective desire. Objective evidence of intent includes: business plans, product development timelines, marketing strategies, manufacturing contracts, website development. Lane Ltd. v. Jackson International Trading Co., 33 USPQ2d 1351 (TTAB 1994).

#### §44(d) — Foreign Priority

- [ ] The applicant has filed a trademark application in a qualifying foreign country
- [ ] The U.S. application is filed within **6 months** of the foreign filing date
- [ ] **Certified copy** of the foreign application (or a certified status copy) must be submitted
- [ ] The applicant is a national of or domiciled in a country that is party to the Paris Convention or a bilateral treaty

**When to use §44(d):** The applicant filed first in a foreign country and wants to claim the earlier filing date as the U.S. priority date. Useful for international brand launches.

#### §44(e) — Foreign Registration

- [ ] The applicant owns a valid, existing registration in their country of origin
- [ ] **Certified copy** of the foreign registration must be submitted
- [ ] The foreign registration must be in force and not subject to cancellation
- [ ] No use in U.S. commerce is required for registration, but use is required for maintenance (§8/§71)

**When to use §44(e):** The applicant already has a registration in their home country and wants to extend protection to the U.S. without proving current U.S. use.

#### §66(a) — Madrid Protocol

- [ ] The applicant holds an International Registration through WIPO designating the United States
- [ ] The application is filed through the Madrid System (not directly with the USPTO)
- [ ] Subject to 5-year dependency on the home registration
- [ ] Examined by the USPTO under the same standards as domestic applications

**When to use §66(a):** The applicant is using the Madrid System for international trademark protection and has designated the U.S.

### 4. International Class(es) and Identification of Goods/Services

- [ ] Correct Nice Classification class(es) selected (see `benny-nice-classification.md` and `benny-class-recommendation.md`)
- [ ] Identification of goods/services is **specific, definite, and clear**
- [ ] For TEAS Plus: identification selected from the ID Manual
- [ ] For TEAS Standard: custom identification follows USPTO conventions
- [ ] Each class has its own identification, specimen (if §1(a)), and fee
- [ ] Multi-class applications: verify all classes are correct and complete

### 5. Specimen of Use (§1(a) Applications Only)

**For goods (Classes 1–34):**
- [ ] Shows the mark affixed to or directly associated with the goods
- [ ] Shows the mark at the point of sale (or a purchasing mechanism for web specimens)
- [ ] JPG or PDF format, clear and legible
- [ ] If a webpage: includes the URL and access/print date
- [ ] One specimen per class
- [ ] Mark on the specimen matches (or substantially matches) the mark in the drawing

**For services (Classes 35–45):**
- [ ] Shows the mark used in the sale or advertising of the services
- [ ] Associates the mark with the specific services identified
- [ ] JPG or PDF format, clear and legible
- [ ] If a webpage: includes the URL and access/print date
- [ ] One specimen per class

**Specimen red flags to avoid:**
- [ ] NOT a digitally created mockup or prototype
- [ ] NOT mere advertising without a purchasing mechanism (for goods)
- [ ] NOT an internal document, invoice alone, or press release alone
- [ ] NOT a specimen showing a materially different mark than what was filed
- [ ] NOT a specimen from before the date of first use in commerce stated in the application

### 6. Declaration/Verification

- [ ] Signed by the applicant (or an authorized person)
- [ ] For individuals: signed by the individual
- [ ] For entities: signed by a person with authority to bind the entity (officer, managing member, partner, etc.)
- [ ] Signatory's name, title, and date included
- [ ] Declaration includes the standard statements required by 37 CFR §2.20:
  - Belief in entitlement to use the mark
  - No knowledge of others' right to use the mark
  - Facts are true to the best of the signatory's knowledge
  - Understanding that willful false statements are punishable

### 7. Filing Fee

| Filing Type | Fee Per Class (2025–2026) |
|-------------|--------------------------|
| TEAS Plus | $250 |
| TEAS Standard | $350 |
| Additional class (TEAS Plus) | $250 |
| Additional class (TEAS Standard) | $350 |

- [ ] Fee calculated: Number of classes x per-class fee
- [ ] Payment method: credit card, deposit account, or EFT
- [ ] Fee is non-refundable, even if the application is refused or abandoned

## Common Filing Errors and How to Avoid Them

### Error 1: Wrong Entity Type

**Problem:** Applicant states "individual" but is actually an LLC, or states "corporation" but is an LLC.
**Prevention:** Confirm the exact legal entity type with the applicant. Check state filings if unsure. An LLC is NOT a corporation.

### Error 2: Wrong Filing Basis

**Problem:** Applicant selects §1(a) (use in commerce) but has not actually used the mark yet.
**Prevention:** Confirm actual current use before selecting §1(a). If the applicant hasn't started using the mark, use §1(b).

### Error 3: Dates of First Use Are Incorrect

**Problem:** The date of first use in commerce predates when the applicant actually started selling/providing services under the mark.
**Prevention:** Ask for specific evidence of first use (invoices, website launch dates, shipping records). Err on the side of a later (more conservative) date.

### Error 4: Identification of Goods/Services Is Too Broad

**Problem:** "Computer software" or "clothing" without further specification triggers an Office action.
**Prevention:** Always specify the type, purpose, or field of use: "Downloadable computer software for project management" or "T-shirts, sweatshirts, and athletic leggings."

### Error 5: Wrong Class

**Problem:** SaaS filed in Class 9 (should be 42), or downloadable software filed in Class 42 (should be 9).
**Prevention:** Use `benny-nice-classification.md` and `benny-class-recommendation.md` to verify classification.

### Error 6: Unacceptable Specimen

**Problem:** Specimen is a mockup, does not show the mark, or does not include a purchasing mechanism for goods.
**Prevention:** Use the specimen checklist above. For web specimens, always include the URL and access date.

### Error 7: Mark in Drawing Does Not Match Specimen

**Problem:** The application claims a standard character mark but the specimen shows a significantly stylized version, or vice versa.
**Prevention:** Ensure consistency between the drawing/application and the specimen. A standard character claim covers all presentations, so stylization in the specimen is acceptable for a standard character filing.

### Error 8: Missing Domicile Address

**Problem:** The applicant provides only a P.O. Box or no address at all.
**Prevention:** Obtain the applicant's actual physical domicile address (street address required).

### Error 9: No U.S. Attorney for Foreign Applicant

**Problem:** Foreign-domiciled applicants filing without a U.S.-licensed attorney.
**Prevention:** Confirm the applicant's domicile. If outside the U.S., a U.S.-licensed attorney must be appointed.

### Error 10: TEAS Plus With Non-ID-Manual Description

**Problem:** Filing as TEAS Plus but using a custom (free-form) identification that is not in the ID Manual.
**Prevention:** Search the ID Manual first. If no match, file as TEAS Standard.

## Pre-Filing Due Diligence Checklist

Before filing, ensure these steps are complete:

- [ ] **Trademark clearance search** conducted (see `benny-conflict-search.md`)
- [ ] **Distinctiveness assessment** completed (see `benny-distinctiveness-assessment.md`)
- [ ] **Class recommendation** confirmed (see `benny-class-recommendation.md`)
- [ ] **No identified blocking marks** (or strategy in place if moderate-risk marks found)
- [ ] **Applicant identity confirmed** (correct legal name, entity type, domicile)
- [ ] **Mark finalized** (applicant has settled on the final version of the mark)
- [ ] **Specimen available** (for §1(a) filings) or **bona fide intent documented** (for §1(b))
- [ ] **Budget confirmed**: filing fees + potential Office action response fees + maintenance fees

## Post-Filing Timeline

| Event | Expected Timing |
|-------|----------------|
| Filing receipt and serial number | Immediate (electronic filing) |
| Assignment to examining attorney | 2–4 months |
| First Office action (if issues found) | 3–7 months |
| Approval for publication (if no issues) | 3–7 months |
| Publication in Official Gazette | 4–8 months |
| Opposition period (30 days) | After publication |
| Registration (§1(a)) or NOA (§1(b)) | 5–12 months (if no opposition) |
| Statement of Use deadline (§1(b)) | 6 months from NOA (extendable to 36 months) |
| Section 8 affidavit | Between 5th and 6th year post-registration |
| Section 9 renewal | Every 10 years |
