# TEAS Filing Guide -- Trademark Electronic Application System

> Sources: Examination Guide 1-25 (January 2025), TEAS Plus Application Form (v6.1), TEAS Standard Application Form (v7.5), 37 CFR Part 2
> Processed for Atlas UX Knowledge Base

---

## Overview

TEAS (Trademark Electronic Application System) is the USPTO's mandatory electronic filing system for trademark applications and most post-registration filings. As of January 18, 2025, the fee structure was simplified from the former TEAS Plus / TEAS Standard distinction to a single base electronic application with additional fees for certain deficiencies.

---

## Fee Structure (Effective January 18, 2025)

### Base Application Fee
| Filing Method | Fee per Class |
|--------------|---------------|
| Electronic (TEAS) | $350 |
| Paper filing | $750 |

### Additional Fees (Assessed During Examination)
| Fee Type | Amount | When Assessed |
|----------|--------|---------------|
| Insufficient information fee | $100/class | When application lacks information required under section 2.22(a)(1)-(20) |
| Free-form text identification fee | $200/class | When goods/services identification is not selected from the USPTO ID Manual |
| Excess character identification fee | $200 per 1,000 characters beyond first 1,000 | When free-form text identification exceeds 1,000 characters per class |

### Key Fee Notes
- The base $350/class fee applies to ALL electronic applications regardless of how goods/services are identified
- Additional fees are assessed via Office Action if the examiner determines deficiencies exist
- Applicants can AVOID the $200 free-form text fee by selecting goods/services descriptions from the USPTO ID Manual
- Applicants can AVOID the $100 insufficient information fee by providing all 20 items listed in section 2.22(a)
- The excess character fee applies per 1,000-character increment (e.g., 2,001 characters = $400 additional)

### Historical Fee Context (Pre-January 18, 2025)
- TEAS Plus: $250/class (required ID Manual descriptions and all information upfront)
- TEAS Standard: $350/class (allowed free-form descriptions)
- These categories are now eliminated; a single $350 base fee applies

### Transition Rules (Applications Filed Before January 18, 2025)
- **Pending TEAS Plus applications**: Continue under old TEAS Plus rules and $250 fee
  - If TEAS Plus requirements not met, additional $125/class processing fee still applies
- **Pending TEAS Standard applications**: Continue under old rules at $350 fee
- **New applications filed on/after January 18, 2025**: Subject to new fee structure regardless of when goods/services were conceived

## Application Form Sections (Step-by-Step)

### 1. Instruction Page
- Navigation breadcrumb: Instruction > Applicant > Mark > Goods/Services/Filing Basis > Attorney/Dom. Rep./Correspondence > Fee/Signature
- Form type selection (now unified post-Jan 2025)

### 2. Applicant Information
**Required fields (marked with *):**
- **Owner of Mark**: Full legal name of the individual or entity that owns the mark
- **Entity Type**: Individual, Corporation, LLC, Partnership, Joint Venture, Sole Proprietorship, Trust, Estate, etc.
- **State/Country of Citizenship** (individuals) or **State/Country of Incorporation/Organization** (entities)
- **Street Address**: Physical domicile address
- **City, State, Country, Zip/Postal Code**
- **Email Address**

**Optional fields:**
- Phone Number
- Fax Number
- Website URL

> **Compliance Note**: Foreign-domiciled applicants MUST designate a U.S.-licensed attorney OR appoint a domestic representative.

### 3. Mark Information
**Mark Type Options:**

#### Standard Character Mark
- Letters, numbers, and/or punctuation only
- No claim to any particular font, style, size, or color
- Broadest protection -- covers the mark in any font/style

#### Special Form Mark (Design/Stylized)
- Mark includes design elements, stylization, or color
- Must upload a JPG/PDF image of the mark
- If **color claimed**: Must include:
  - Color claim statement identifying colors
  - Description of mark specifying where each color appears

#### Sound Mark
- Must upload MP3/WAV audio file
- Must include description of the sound

**Additional Mark Statements (if applicable):**
- **Translation**: English translation of any non-English wording
- **Transliteration**: English transliteration of non-Latin characters
- **Claimed Prior Registration(s)**: Active prior U.S. registration numbers
- **Consent (Name/Likeness)**: If mark contains a living person's name/likeness/signature
- **Concurrent Use Claim**
- **Disclaimer**: "No claim is made to the exclusive right to use ___ apart from the mark as shown"
- **Stippling as Feature of Mark**: Statement that stippling is a feature, not indicating color
- **Significance of Mark**: Meaning/significance of wording in the mark

### 4. Goods/Services Identification

#### Using the ID Manual (Recommended -- Avoids $200 Free-Form Fee)
- Search the USPTO Acceptable Identification of Goods and Services Manual
- Select pre-approved descriptions
- Each description is pre-classified by International Class number

#### Free-Form Text Entry
- Available but triggers $200/class additional fee
- Must be specific enough to clearly identify the nature of goods/services
- Exceeding 1,000 characters per class triggers additional $200 per 1,000-character increment
- Examiner may further require amendment to acceptable ID

#### International Classification
- Goods: Classes 1-34
- Services: Classes 35-45
- Each class requires a separate fee
- Multiple items can be listed within the same class

### 5. Filing Basis

#### Section 1(a) -- Use in Commerce
"Actually using mark in commerce now"
- **Required fields:**
  - Date of First Use Anywhere (MM/DD/YYYY)
  - Date of First Use in Commerce (MM/DD/YYYY)
  - Specimen showing mark as used in commerce (JPG/PDF)
  - Description of specimen

> **Specimen Requirements for Goods**: Tags, labels, containers, packaging, product displays, website product pages showing mark with purchasing capability
> **Specimen Requirements for Services**: Advertising, marketing materials, business signage, website pages advertising the services
> **WARNING**: The specimen must show the mark as ACTUALLY USED in commerce. A scan of a complete business card is acceptable as a specimen but would not be acceptable as the mark image.

#### Section 1(b) -- Intent to Use
"No use of mark yet, intending to use"
- Applicant has a bona fide intention to use the mark in commerce
- **No specimen required** at time of filing
- Must later file either:
  - **Amendment to Allege Use** (before approval for publication), OR
  - **Statement of Use** (after Notice of Allowance)
- Additional fee required with Statement of Use ($150/class)
- Cannot claim BOTH 1(a) and 1(b) for the SAME goods/services

> **WARNING**: Selecting 1(b) means the applicant must (1) begin using the mark in commerce, AND (2) file an Allegation of Use with a specimen and additional fee, BEFORE the mark can register.

#### Section 44(d) -- Foreign Priority
"Foreign application exists for same goods/services"
- **Required fields:**
  - Country of Foreign Filing
  - Foreign Application Number (max 19 characters)
  - Date of Foreign Filing (MM/DD/YYYY)
- Provides a priority date but is NOT a basis for registration by itself
- Must indicate whether applicant intends to rely on 44(e) (suspend for foreign registration) or has another basis (1(a), 1(b))

#### Section 44(e) -- Foreign Registration
"Foreign registration exists for same goods/services"
- **Required fields:**
  - Country of Foreign Registration
  - Foreign Registration Number (max 19 characters)
  - Foreign Registration Date (MM/DD/YYYY)
  - Date Foreign Registration Renewed (if applicable)
  - Expiration Date of Foreign Registration (MM/DD/YYYY)
  - Attached copy of Foreign Registration Certificate and/or Proof of Renewal/Translation

#### Section 66(a) -- Madrid Protocol
- NOT filed directly through TEAS
- Extension of protection from International Bureau to the U.S.
- Application automatically created when International Bureau transmits request

#### Multiple Bases
- Multiple filing bases may be selected for different goods/services
- Cannot claim BOTH 1(a) and 1(b) for identical goods/services
- May claim 1(a) for some items and 1(b) for others within the same class

### 6. Attorney / Domestic Representative / Correspondence

#### Attorney Information (if represented)
**Required fields:**
- Attorney Name
- Year of Admission
- U.S. State/Commonwealth/Territory of bar membership
- Membership Number (up to 40 alphanumeric characters)
- Attestation: "The attorney of record is an active member in good standing of the bar of the highest court of a U.S. state, the District of Columbia, or any U.S. Commonwealth or territory"
- Street Address, City, State, Country, Zip/Postal Code
- Email Address (mandatory; authorize email communication)

**Optional fields:**
- Individual Attorney Docket/Reference Number (max 12 characters)
- Other Appointed Attorney(s)
- Firm Name
- Internal Address
- Phone Number, Fax Number

#### Domestic Representative (if no attorney and foreign applicant)
- Required for foreign applicants without U.S. attorney
- Name and U.S. address of person authorized to receive service of process

#### Correspondence Information
**Required fields:**
- Name, Street Address, City, State, Country, Zip/Postal Code
- Email Address (must authorize electronic communication)

> **Critical**: Email authorization is MANDATORY. Failure to authorize email communication will result in additional processing fees. All USPTO notices and Office Actions are delivered via email.

### 7. Fee / Signature

#### Fee Information
- Displays number of classes, fee per class, and total fee due
- Payment options (after validation): Credit/debit card, automated deposit account, Electronic Funds Transfer

#### Signature Methods
1. **Sign directly** -- Electronic signature between forward slashes (e.g., /john doe/)
2. **Email Text Form to second party** -- For situations where signer is different from filer
3. **Handwritten pen-and-ink signature** -- Print, sign, scan, and upload

#### Declaration (Required)
Must acknowledge by checking boxes:
- **Section 1(a) basis**: Signatory believes applicant is owner; mark is in use in commerce; specimen shows mark as used; facts recited are accurate
- **Section 1(b) basis**: Signatory believes applicant is entitled to use mark; applicant has bona fide intention to use; facts recited are accurate
- **Additional declarations**:
  - No other persons have right to use the mark in identical/confusingly similar form (exclusive use claim)
  - Allegations and factual contentions have evidentiary support
  - Warning about willful false statements (18 U.S.C. 1001)

**Required signatory fields:**
- Signature (between forward slashes)
- Date Signed (MM/DD/YYYY)
- Signatory's Name
- Signatory's Position (Owner, Holder, President, Attorney of Record, etc.)

### 8. Validation Page
- Confirms: "You completed all mandatory fields and successfully validated the form. It has NOT been filed to the USPTO at this point."
- **STEP 1**: Review Application Data (Input, Mark, Specimen, Consent(s), XML File, Text Form)
- **STEP 2**: Confirm email address for acknowledgment
- **STEP 3**: Save Form (downloads .obj/.xml file for later submission)
- **STEP 4**: Read Important Notice (fee is non-refundable; all information becomes public record)
- **STEP 5**: Click Pay/Submit to proceed to payment

> **WARNING**: Once you click Pay/Submit, you CANNOT return to the form. You must complete the payment process within 30 minutes. Credit card payments not available 2-6 AM Sunday Eastern Time.

### 9. Payment
- USPTO Pay portal
- Options: Sign in (USPTO Financial Manager) or Pay as Guest
- Credit/Debit Card: Visa, MasterCard, Discover, American Express
- Card Number, Security Code, Name on Card, Expiration Date
- Billing Address required

## Common Filing Errors and How to Avoid Them

### 1. Incorrect Identification of Goods/Services
- **Error**: Using vague or overly broad descriptions (e.g., "merchandise," "various goods")
- **Fix**: Use the USPTO ID Manual for pre-approved descriptions; be specific about the nature of goods/services

### 2. Improper Specimens
- **Error**: Submitting a drawing/mockup instead of actual use specimen; submitting website pages that don't show the mark with the goods/services
- **Fix**: Ensure specimen shows mark as ACTUALLY USED in ordinary course of trade

### 3. Incorrect Entity Type
- **Error**: Filing as individual when business entity owns the mark; incorrect state of incorporation
- **Fix**: Verify legal ownership before filing; the applicant MUST be the legal owner of the mark

### 4. Missing or Incorrect Filing Basis
- **Error**: Claiming 1(a) when not yet using the mark; claiming 44(d) outside the 6-month priority window
- **Fix**: Use 1(b) if not yet using the mark; verify foreign filing dates for 44(d) claims

### 5. Inadequate Mark Description
- **Error**: Failing to describe design elements, colors, or non-Latin characters
- **Fix**: Include color claim statement if colors are claimed; translate all non-English wording; transliterate all non-Latin characters

### 6. Signature Deficiencies
- **Error**: Signature by unauthorized person; missing signatory position/title
- **Fix**: Ensure signer has legal authority (individual owner, corporate officer, authorized attorney); always include title/position

### 7. Late Filings and Missed Deadlines
- **Error**: Missing 6-month deadline for Statement of Use; missing 3-month Office Action response deadline
- **Fix**: Calendar all deadlines immediately; request extensions BEFORE deadlines expire

### 8. Failure to Authorize Email Communication
- **Error**: Not checking the email authorization checkbox
- **Fix**: ALWAYS authorize email communication to avoid additional processing fees and ensure receipt of all USPTO correspondence

## Post-Filing: What Happens Next

1. **Filing Receipt**: Electronic acknowledgment within 24 hours
2. **Assigned to Examining Attorney**: Typically 3-8 months after filing
3. **Office Action** (if issues found): Must respond within 3 months (extendable to 6)
4. **Approval for Publication**: If no issues or all resolved
5. **Publication in Official Gazette**: 30-day opposition period
6. **Notice of Allowance** (1(b) only): Must file SOU within 6 months (extendable to 36)
7. **Registration Certificate**: Issues after all requirements met
8. **Post-Registration Maintenance**: Section 8 declaration (years 5-6), Section 9 renewal (year 10)
