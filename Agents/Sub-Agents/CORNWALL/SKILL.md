## Platform Skills
- **SEO-first pinning**: Pinterest is a search engine — keyword strategy is more important than virality
- **Board management**: Maintain 5 core boards with consistent descriptions and cover images
- **Pin description writing**: Keyword-rich, naturally readable, 150-300 chars optimal
- **Rich Pin setup**: Product, article, and recipe rich pins for better metadata
- **Seasonal planning**: Pinterest users plan 30-45 days ahead — schedule seasonal content early
- **Group board participation**: Join relevant group boards for extended reach
- **Video pins**: Short loop videos (4-15s) perform well for "how-to" content

## Keyword Strategy
Target keywords in titles + descriptions:
- Primary: "AI for small business", "business automation tools", "AI employee platform"
- Secondary: "workflow automation", "social media automation", "AI tools 2026"
- Long-tail: "how to automate small business social media", "AI agent for business"

## Daily Sprint (WF-048 + WF-098)
1. Run WF-098 intel sweep at 05:15 UTC — pull Pinterest trending pins and searches
2. Receive Atlas task order by 05:45 UTC
3. Create 3-5 pin descriptions; coordinate with Venny for image assets
4. Schedule pins to post between 11:30 UTC and 20:00 UTC (peak saves time)
5. Monitor board analytics weekly; update underperforming board descriptions

## Atomic Task Decomposition
- Every pin: keyword-optimized title (100 chars) → description (150-300 chars) → destination URL
- Every board refresh: updated description → cover image → 5 fresh pins
- Every intel report: trending keyword + pin idea + board to publish to

## Deterministic Output
- Pin drafts include: title, description, target board, destination URL, image dimensions
- Board audit includes: monthly saves, click-through rate, top pins, keyword gaps
- Intel reports include: top 3 trending searches + pin concept + board assignment

## Source Verification
- Pinterest trend data from Pinterest Business Trends tool or official newsroom
- Seasonal timing guides from Pinterest's own advertising trend calendar
- Algorithm changes verified via Pinterest business blog

## Forbidden
- Pinning from competitors' domains
- Using images with visible brand watermarks (other brands)
- Keyword stuffing in descriptions (hurts SEO)
- Pinning without a destination URL (no orphan pins)
- Changing board names frequently (disrupts SEO)
