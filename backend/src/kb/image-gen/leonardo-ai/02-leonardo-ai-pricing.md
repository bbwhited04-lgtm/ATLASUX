# Leonardo.ai Pricing Guide

## Overview

Leonardo.ai uses a token-based pricing system where tokens serve as the platform currency for all generation activities. Tokens are consumed for image generation, video creation, upscaling, background removal, and other post-processing tasks. The number of tokens consumed per action varies based on computational intensity and feature tier.

## Pricing Tiers

### Free Plan
- **Cost:** $0/month
- **Tokens:** 150 fast tokens per day (reset every 24 hours)
- **Features:** Access to base models, standard generation, community models
- **Limitations:** No private generations, limited model access, queue priority is lowest, watermarked outputs
- **Best for:** Casual exploration, evaluating the platform before committing

### Apprentice Plan
- **Cost:** $12/month (or $10/month billed annually)
- **Tokens:** 8,500 tokens per month
- **Features:** Private generations, priority queue, access to all models including Phoenix and Alchemy, no watermarks
- **Best for:** Hobbyists, freelancers with moderate generation needs

### Artisan Plan
- **Cost:** $30/month (or $24/month billed annually)
- **Tokens:** 25,000 tokens per month
- **Features:** Everything in Apprentice plus higher concurrency limits, faster generation speeds, custom model training
- **Best for:** Professional creators, small studios, regular production use

### Maestro Plan
- **Cost:** $60/month (or $48/month billed annually)
- **Tokens:** 60,000 tokens per month
- **Features:** Everything in Artisan plus maximum concurrency, fastest queue priority, highest resolution options
- **Best for:** Studios, agencies, and power users with high-volume production needs

| Standard image (no Alchemy) | 1-2 tokens per image |

*Note: Token costs vary based on resolution, number of images per generation, and specific model/settings used. Use Leonardo's built-in Pricing Calculator for exact costs.*

## Token Top-Ups

All paid plans support manual token top-ups when monthly allocations run out. Pricing varies by plan tier:

| Top-Up Amount | Basic Plan | Standard Plan | Pro Plan |
|---------------|-----------|---------------|---------|
| 5,000 tokens | $15.00 | $11.00 | $8.00 |

Higher-tier plans receive better per-token pricing on top-ups, incentivizing commitment to larger monthly plans.

## API Pricing

Leonardo launched its Creative Engine API in 2026. API access requires a paid subscription and uses the same token system as the web interface. Key API pricing details:

- **Enterprise API:** Starts at approximately $0.002 per standard generation
- **Token consumption:** Same rates as the web platform (standard vs. Alchemy)
- **Rate limits:** 5 concurrent requests for standard API keys, 10 concurrent generation jobs
- **Enterprise plans:** Custom rate limits, dedicated support, volume pricing available on request

API top-up costs follow the same tiered structure as manual top-ups, with higher-tier plans receiving better per-token rates.

## Recommendation for Atlas UX

For Atlas UX agent image generation needs, the **Artisan plan** ($24/month annual) provides the best balance of token volume (25,000/month) and cost efficiency. At Alchemy-enhanced quality, this yields approximately 1,500-3,000 images per month. For burst production periods, token top-ups at the standard rate ($11/5,000 tokens) provide flexible overflow capacity. API access enables programmatic generation through the agents without manual intervention.
