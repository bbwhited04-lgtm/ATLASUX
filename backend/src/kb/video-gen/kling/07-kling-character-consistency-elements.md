## Content

### The Elements System

Kling's Elements system is the core feature for maintaining character consistency across video generations. Users can:

1. Upload 1 to 4 reference images
2. Select specific subjects (people, animals, objects, scenes)
3. Describe actions and interactions
4. Generate dynamic, high-quality videos with consistent character appearance

This feature provides precise control over character appearances across multiple shots, in different situations and locations.

### How Elements Works

**Reference Image Selection:**
The most reliable way to stabilize a character is to use the same reference image for every generation. This becomes the visual anchor the model uses to infer:
- Facial features and proportions
- Clothing and accessories
- Body type and posture
- Hair style and color
- Overall character identity

**Multi-Image References:**
When using multiple reference images:
- Provide different angles of the same character
- Include full body and close-up shots
- Keep clothing and styling consistent across reference images
- The model uses all references to build a 3D understanding of the character

### Elements 3.0 Omni (Advanced)

For even stronger consistency, Elements 3.0 Omni offers:

**Video Character References:**
- Upload a 3-8 second video clip of your character
- The model extracts both visual traits and voice characteristics simultaneously
- More accurate than static images for capturing movement style and mannerisms

**Voice Binding:**
- Upload multiple images + a voice clip (3+ seconds)
- The model binds visual identity and voice together
- Lip sync becomes more accurate
- Expressions feel more alive
- Storytelling gains emotional depth

### Character Consistency Across Shots

Identity stays consistent through:
- **Camera angle changes**: Close-up to wide shot, front to profile
- **Scene transitions**: Indoor to outdoor, different locations
- **Character interactions**: Multiple characters in the same frame
- **Lighting changes**: Day to night, different color temperatures
- **Action sequences**: Running, sitting, gesturing

### Best Practices for Maximum Consistency

1. **Use a single primary reference image** for all generations with that character
2. **Keep prompts consistent** with character descriptions across generations
3. **Use the same model version** for all clips in a sequence
4. **Describe clothing explicitly** even when using reference images
5. **Avoid conflicting descriptions** -- don't describe blonde hair if the reference shows dark hair
6. **Generate in batches** -- multiple clips in one session tend to be more consistent
7. **Use Elements 3.0 Omni** for highest consistency with video references

### Multi-Character Scenes

When working with multiple characters:
- Assign each character a distinct Element reference
- Use clear spatial descriptions ("the woman on the left," "the man standing behind the desk")
- Keep character interactions specific ("she hands him the coffee cup")
- The model maintains distinct identities even in group shots

### Style References

Beyond character consistency, Elements supports style references:
- Upload a reference image for overall visual style
- Match color grading, lighting mood, and aesthetic
- Combine style reference with character reference for maximum control
- Useful for maintaining brand consistency across marketing videos
