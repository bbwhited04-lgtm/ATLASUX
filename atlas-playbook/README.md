# Atlas UX Playbook

10 specialized agents + 6 strategic frameworks, adapted from
[product-architect](https://github.com/ankitjha67/product-architect) (MIT License)
and rewritten entirely for Atlas UX.

Start with `SMART-LOADER.md` — it routes requests to the right files.
