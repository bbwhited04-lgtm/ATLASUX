## 1. Number Theory

### Fundamental Theorems
- **Fundamental Theorem of Arithmetic**: Every integer > 1 is either prime or a unique product of primes
- **Fermat's Little Theorem**: If p is prime and gcd(a,p)=1, then a^(p-1) ≡ 1 (mod p)
- **Euler's Theorem**: If gcd(a,n)=1, then a^φ(n) ≡ 1 (mod n)
- **Chinese Remainder Theorem**: System of congruences x ≡ a_i (mod n_i) has unique solution mod N = ∏n_i when n_i are pairwise coprime
- **Quadratic Reciprocity**: For odd primes p,q: (p/q)(q/p) = (-1)^((p-1)/2 · (q-1)/2)
- **Wilson's Theorem**: (p-1)! ≡ -1 (mod p) iff p is prime
- **Legendre's Formula**: v_p(n!) = Σ_{i=1}^∞ ⌊n/p^i⌋

### Key Functions
- **Euler's totient**: φ(n) = n · ∏(1 - 1/p) for prime p | n
- **Möbius function**: μ(n) = (-1)^k if n is squarefree (k prime factors), else 0
- **Möbius inversion**: If f(n) = Σ_{d|n} g(d), then g(n) = Σ_{d|n} μ(n/d)f(d)

### Important Sequences & Constants
- Fibonacci: F(n) = F(n-1) + F(n-2), closed form: F(n) = (φ^n - ψ^n)/√5 where φ = (1+√5)/2
- Catalan numbers: C(n) = (2n)!/((n+1)!n!) = C(2n,n)/(n+1)
- Bernoulli numbers: B_0=1, B_1=-1/2, B_2=1/6, B_4=-1/30, B_6=1/42
- Partition function p(n): Hardy-Ramanujan asymptotic: p(n) ~ (1/(4n√3)) · e^(π√(2n/3))

## 2. Abstract Algebra

### Group Theory
- **Lagrange's Theorem**: |H| divides |G| for subgroup H ≤ G
- **Sylow Theorems**:
  1. If p^k divides |G|, G has a subgroup of order p^k
  2. All Sylow p-subgroups are conjugate
  3. n_p ≡ 1 (mod p) and n_p divides |G|/p^k
- **Cauchy's Theorem**: If prime p divides |G|, G has an element of order p
- **Classification of Finite Abelian Groups**: Every finite abelian group is isomorphic to a direct product of cyclic groups of prime power order
- **Burnside's Lemma**: |X/G| = (1/|G|) Σ_{g∈G} |X^g| (counting orbits)

### Ring Theory
- **Ring**: Set with +, × where (R,+) is abelian group, × is associative, distributive
- **Ideal**: I ⊆ R such that RI ⊆ I and IR ⊆ I
- **PID** (Principal Ideal Domain): Every ideal is generated by a single element
- **UFD** (Unique Factorization Domain): Every non-zero non-unit factors uniquely into irreducibles
- **Eisenstein's Criterion**: f(x) = Σa_i x^i is irreducible over Q if prime p divides all a_i except a_n, p divides a_0, p^2 does not divide a_0

### Field Theory & Galois Theory
- **Splitting field**: Smallest field extension where polynomial factors completely
- **Galois group**: Gal(E/F) = automorphisms of E fixing F
- **Fundamental Theorem of Galois Theory**: Subgroups of Gal(E/F) correspond to intermediate fields
- **Insolvability of quintic**: S_5 is not solvable → general degree ≥ 5 polynomial not solvable by radicals
- **Finite fields**: GF(p^n) exists and is unique for every prime p and positive integer n

## 3. Analysis

### Real Analysis
- **Bolzano-Weierstrass**: Every bounded sequence in R^n has a convergent subsequence
- **Heine-Borel**: S ⊆ R^n is compact iff closed and bounded
- **Intermediate Value Theorem**: Continuous f on [a,b] with f(a) < c < f(b) has f(x) = c for some x ∈ (a,b)
- **Mean Value Theorem**: f'(c) = (f(b)-f(a))/(b-a) for some c ∈ (a,b)
- **Taylor's Theorem**: f(x) = Σ_{k=0}^{n} f^(k)(a)/k! · (x-a)^k + R_n(x)
- **Weierstrass Approximation**: Every continuous function on [a,b] can be uniformly approximated by polynomials

### Complex Analysis
- **Cauchy's Integral Formula**: f(a) = (1/2πi) ∮ f(z)/(z-a) dz
- **Residue Theorem**: ∮ f(z)dz = 2πi Σ Res(f, a_k)
- **Liouville's Theorem**: Bounded entire function is constant
- **Maximum Modulus Principle**: |f(z)| on a domain attains max only on boundary
- **Fundamental Theorem of Algebra**: Every non-constant polynomial over C has a root
- **Laurent Series**: f(z) = Σ_{n=-∞}^{∞} a_n(z-c)^n in annular region

### Measure Theory
- **Lebesgue Dominated Convergence**: If |f_n| ≤ g (integrable) and f_n → f a.e., then ∫f_n → ∫f
- **Monotone Convergence**: If 0 ≤ f_n ↑ f, then ∫f_n ↑ ∫f
- **Fatou's Lemma**: ∫ liminf f_n ≤ liminf ∫f_n
- **Fubini's Theorem**: ∫∫ f dμdν = ∫∫ f dνdμ for σ-finite measures when f is integrable

## 4. Topology

### Point-Set Topology
- **Hausdorff space**: Any two distinct points have disjoint open neighborhoods
- **Connected**: Cannot be partitioned into two non-empty open sets
- **Path-connected ⇒ Connected** (converse fails in general)
- **Compact**: Every open cover has a finite subcover
- **Tychonoff's Theorem**: Product of compact spaces is compact
- **Urysohn's Lemma**: In a normal space, disjoint closed sets can be separated by a continuous function

### Algebraic Topology
- **Fundamental group**: π_1(S^1) = Z, π_1(T^2) = Z×Z
- **Brouwer Fixed Point**: Every continuous f: D^n → D^n has a fixed point
- **Borsuk-Ulam**: Every continuous f: S^n → R^n has f(x) = f(-x) for some x
- **Euler Characteristic**: χ = V - E + F (for polyhedra), χ(S^2) = 2, χ(T^2) = 0
- **Hurewicz Theorem**: First non-trivial homotopy group equals first non-trivial homology group
- **Mayer-Vietoris**: Long exact sequence relating homology of union to parts

## 5. Combinatorics

### Counting Principles
- **Binomial coefficient**: C(n,k) = n!/(k!(n-k)!)
- **Multinomial coefficient**: n!/(k_1! k_2! ... k_r!)
- **Stars and Bars**: C(n+k-1, k-1) ways to place n identical objects in k bins
- **Inclusion-Exclusion**: |A₁ ∪ ... ∪ Aₙ| = Σ|Aᵢ| - Σ|Aᵢ∩Aⱼ| + ...
- **Derangements**: D(n) = n! Σ_{k=0}^{n} (-1)^k/k!
- **Stirling numbers (2nd kind)**: S(n,k) = number of partitions of [n] into k non-empty subsets

### Graph Theory
- **Euler's formula (planar)**: V - E + F = 2
- **Handshaking lemma**: Σ deg(v) = 2|E|
- **Kuratowski's Theorem**: G is planar iff no subdivision of K_5 or K_{3,3}
- **Four Color Theorem**: Every planar graph is 4-colorable
- **Ramsey numbers**: R(3,3)=6, R(4,4)=18, R(5,5) is unknown (43-48)
- **Chromatic polynomial**: P(G,k) counts proper k-colorings
- **König's Theorem**: In bipartite graphs, maximum matching = minimum vertex cover

### Generating Functions
- Ordinary: Σ a_n x^n — for counting without regard to order
- Exponential: Σ a_n x^n/n! — for labeled structures
- **Catalan OGF**: C(x) = (1 - √(1-4x))/(2x)
- **Partition OGF**: Π_{k≥1} 1/(1-x^k)

## 6. Linear Algebra

### Core Results
- **Rank-Nullity**: dim(ker T) + dim(im T) = dim V
- **Spectral Theorem**: Real symmetric matrix has orthogonal eigenbasis
- **Cayley-Hamilton**: Every square matrix satisfies its own characteristic polynomial
- **Jordan Normal Form**: Every matrix over C is similar to a Jordan form
- **Singular Value Decomposition**: A = UΣV^T for any m×n matrix
- **Determinant**: det(AB) = det(A)det(B), det(A^T) = det(A)

### Matrix Properties
- Trace: tr(AB) = tr(BA), tr(A) = Σ eigenvalues
- Eigenvalues of A^(-1) are 1/λ_i
- Eigenvalues of A^k are λ_i^k
- Positive definite: all eigenvalues > 0, equivalently x^TAx > 0 for all x ≠ 0
- Normal matrix: AA* = A*A (unitarily diagonalizable)

## 7. Probability & Statistics

### Distributions
- **Binomial**: P(X=k) = C(n,k)p^k(1-p)^{n-k}, E[X]=np, Var=np(1-p)
- **Poisson**: P(X=k) = λ^k e^{-λ}/k!, E[X]=Var=λ
- **Normal**: f(x) = (1/√(2πσ²))exp(-(x-μ)²/(2σ²))
- **Exponential**: f(x) = λe^{-λx}, E[X]=1/λ, memoryless
- **Chi-squared**: Sum of k squared standard normals, df=k
- **Beta**: f(x) = x^{α-1}(1-x)^{β-1}/B(α,β)
- **Gamma**: f(x) = x^{k-1}e^{-x/θ}/(Γ(k)θ^k)

### Key Theorems
- **Law of Large Numbers**: X̄_n → μ as n → ∞
- **Central Limit Theorem**: √n(X̄_n - μ)/σ → N(0,1)
- **Bayes' Theorem**: P(A|B) = P(B|A)P(A)/P(B)
- **Markov's Inequality**: P(X≥a) ≤ E[X]/a for X≥0
- **Chebyshev's Inequality**: P(|X-μ|≥kσ) ≤ 1/k²

### Information Theory
- **Entropy**: H(X) = -Σ p(x) log p(x)
- **KL Divergence**: D_KL(P||Q) = Σ P(x) log(P(x)/Q(x))
- **Mutual Information**: I(X;Y) = H(X) - H(X|Y)
- **Data Processing Inequality**: X → Y → Z ⟹ I(X;Z) ≤ I(X;Y)

## 8. Differential Equations

### ODEs
- **Separable**: dy/dx = f(x)g(y) → ∫dy/g(y) = ∫f(x)dx
- **Linear 1st order**: y' + P(x)y = Q(x), integrating factor μ = e^{∫P dx}
- **Characteristic equation**: ay'' + by' + cy = 0 → ar² + br + c = 0
- **Variation of parameters**: y_p = u_1 y_1 + u_2 y_2
- **Picard-Lindelöf**: Existence and uniqueness for Lipschitz continuous f

### PDEs
- **Heat equation**: u_t = α²u_{xx}, fundamental solution: Gaussian
- **Wave equation**: u_{tt} = c²u_{xx}, d'Alembert solution: u = f(x-ct) + g(x+ct)
- **Laplace equation**: ∇²u = 0, maximum principle applies
- **Schrödinger equation**: iℏ∂ψ/∂t = Ĥψ
- **Green's functions**: Solve Lu = f via u = ∫G(x,ξ)f(ξ)dξ

## 9. Category Theory

### Basic Concepts
- **Category**: Objects + morphisms with composition and identity
- **Functor**: Structure-preserving map between categories (objects → objects, morphisms → morphisms)
- **Natural transformation**: Map between functors η: F ⇒ G, with components η_X: F(X) → G(X)
- **Adjunction**: F ⊣ G means Hom(FX,Y) ≅ Hom(X,GY) naturally
- **Yoneda Lemma**: Nat(Hom(A,-), F) ≅ F(A) — natural transformations from representable functor to F are determined by F(A)

### Limits and Colimits
- **Product**: Categorical product generalizes Cartesian product
- **Coproduct**: Categorical coproduct generalizes disjoint union
- **Equalizer/Coequalizer**: Universal arrows making parallel morphisms equal
- **Pullback/Pushout**: Fiber products and amalgamated sums
- **End/Coend**: ∫_c F(c,c) and ∫^c F(c,c) — generalize natural transformations

### Key Results
- **Freyd's Adjoint Functor Theorem**: Conditions for a functor to have a left adjoint
- **Nerve of a category**: Simplicial set encoding the category
- **Kan extensions**: Universal construction extending functors along other functors

## 10. Mathematical Logic

### First-Order Logic
- **Completeness Theorem (Gödel)**: Every valid FOL formula is provable
- **Compactness Theorem**: If every finite subset of Σ has a model, Σ has a model
- **Löwenheim-Skolem**: If Σ has an infinite model, it has models of all infinite cardinalities
- **Gödel's Incompleteness Theorems**:
  1. Any consistent, sufficiently powerful formal system contains undecidable statements
  2. Such a system cannot prove its own consistency

### Computability
- **Church-Turing Thesis**: Computable functions = Turing-computable functions
- **Halting Problem**: Undecidable (diagonalization argument)
- **Rice's Theorem**: All non-trivial semantic properties of programs are undecidable

## 11. Useful Identities & Formulas

### Sums
- Σ_{k=1}^{n} k = n(n+1)/2
- Σ_{k=1}^{n} k² = n(n+1)(2n+1)/6
- Σ_{k=1}^{n} k³ = [n(n+1)/2]²
- Σ_{k=0}^{n} C(n,k) = 2^n
- Σ_{k=0}^{∞} x^k = 1/(1-x) for |x| < 1
- Σ_{k=1}^{∞} 1/k² = π²/6 (Basel problem)

### Approximations
- Stirling: n! ≈ √(2πn)(n/e)^n
- Harmonic numbers: H_n ≈ ln(n) + γ where γ ≈ 0.5772 (Euler-Mascheroni)
- e ≈ 2.71828, π ≈ 3.14159, √2 ≈ 1.41421, φ = (1+√5)/2 ≈ 1.61803

### Inequalities
- AM-GM: (a₁+...+aₙ)/n ≥ (a₁·...·aₙ)^{1/n}
- Cauchy-Schwarz: |⟨u,v⟩|² ≤ ⟨u,u⟩⟨v,v⟩
- Triangle inequality: |a+b| ≤ |a| + |b|
- Jensen's inequality: f(E[X]) ≤ E[f(X)] for convex f
- Hölder's inequality: Σ|a_i b_i| ≤ (Σ|a_i|^p)^{1/p} (Σ|b_i|^q)^{1/q}
